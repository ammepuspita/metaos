Different types of data sources.

Interfaces govern the structure of the package:

  1. Lines with prices are provided by PricesSources.
  2. Each line is processed by SourceLineProcessors.
  3. Who notifies to subscribed FieldProcessors.

Different types of PricesSources are given:

  • TwoOrderSource for source files with two orders: first in symbol, second in time.
  • OrderedSource for source files with order only in time.

FieldProcessors send messages to a subscribed market (using an observer pattern) with names for symbols like "xxx-HIGH", "xxx-LOW", "xxx-OPEN" or "xxx-CLOSE": 

this.sendPriceToMarkets(moment, symbol + "-HIGH", high);
this.sendPriceToMarkets(moment, symbol + "-LOW", low);
this.sendPriceToMarkets(moment, symbol + "-OPEN", open);
this.sendPriceToMarkets(moment, symbol + "-CLOSE", close);
this.sendVolumeToMarkets(moment, symbol, volume);
this.notifyListeners(moment, {symbol});

To simplify notification process, BasicPricesSource abstract class is provided, with methods to notify markets prices, asks, bids and volumes.
